# CHALLENGE  

## build project from docker-compose.yaml 
```
docker-build up --build
```  
## open app in browser => http://localhost:4200


### TESTING CLIENT / DEFAULT ANGULAR Jasmine & Karma

```
cd client
```
```
npm install
```
```
npm test
```

### TESTING API / Jest & Supertest
```
cd api
```
```
yarn
```
```
yarn test
```
